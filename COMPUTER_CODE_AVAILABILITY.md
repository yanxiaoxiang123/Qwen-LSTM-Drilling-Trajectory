# Computer Code Availability

**Name of code/library:** Qwen-LSTM Drilling Trajectory Prediction

**Developer:** Zhongyi Jiang and contributors

**Year first available:** 2026

**Hardware requirements:** The bundled quick test runs on CPU. Full Qwen-LSTM experiments require a CUDA-capable GPU with sufficient memory for a local Qwen3.5-9B checkpoint; the reported experiments used an NVIDIA RTX 3090.

**Program language:** Python 3.10+ and Bash

**Software required:** NumPy, pandas, scikit-learn, PyTorch, and Hugging Face Transformers. See `requirements.txt`.

**Program size:** Less than 1 MB excluding datasets, generated results, and model weights.

**License:** MIT

**Source repository:** https://github.com/yanxiaoxiang123/Qwen-LSTM-Drilling-Trajectory

The repository permits anonymous download and contains individual source files, documentation, a deidentified example dataset, and a CPU quick test. Model weights and full well datasets are not redistributed.
